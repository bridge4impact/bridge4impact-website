# Bridge4Impact Website

Simple React + Vite website for **Bridge4Impact by Günter Schmittberger**.

## Edit later

- Main website text: `src/App.jsx`
- Styling and colors: `src/styles.css`
- Logo: `public/logo.png`

## Cloudflare Pages build settings

- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`

## Local development (optional)

```bash
npm install
npm run dev
```
